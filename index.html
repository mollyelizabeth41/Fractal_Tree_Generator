Fractal Tree Generator

Description:

This generative web application creates a fractal tree that can be controlled by a graphical user interface (GUI). Below is a description of the parameters.

Parameters:

    BranchL: Branch's on the left side of tree
    BranchR: Branch's on the right side of tree
    Size: Size of tree
    Angle: Angle of tree
    StrokeWeight: Weight of tree stroke
    Color: Color of tree
    BColor: Background color
